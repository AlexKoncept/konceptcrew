# Contributing

Merci pour votre intérêt pour KonceptCrew ! Quelques règles rapides pour contribuer :

- Ouvrez une *issue* pour proposer une fonctionnalité ou rapporter un bug.
- Créez une branche pour vos changements et soumettez une Pull Request vers `main`.
- Ne commitez jamais de secrets (clés API, tokens). Utilisez le fichier `.env` local (ne pas commit).
- Pour exécuter le projet localement :

```bash
npm install
npm run dev
```

Si votre change affecte le build, assurez-vous que `npm run build` passe localement.

Merci !
